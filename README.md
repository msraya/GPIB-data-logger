# GPIB-data-logger
A universal GPIB data logger created to log anything from volts to Hz form several instruments.
By now it uses:

   - HP 53131A
   - HP 34401A
   - Keithley 2015
   - HP 35220A
   
It is based in QtPyGraph and PyVISA and can be integrated with py2exe and NSIS Installer.
